# Exp - 4 HOSPITAL MANAGEMENT SYSTEM

# AIM:
To prepare the SRS and design UML diagrams for a Hospital Management System.

# SRS (Procedure):
1. Study the requirements of HMS.
2. Identify actors (patient, doctor, admin, etc.)
3. Write SRS including functional and non-functional requirements.
4. Create UML diagrams using draw.io / StarUML / hand-drawn images.
5. Upload all 6 diagrams.
6. Compile final report with AIM, Procedure, Diagrams, and Result.
